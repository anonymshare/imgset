
	==========================================================
	The Dataset for Personalized Image Emotion Classification
	==========================================================

Each line in imgset.txt corresponding to one image:
owner=<image owner>	id=<image ID>	tag=<image emotion>	url=<image URL>

owner: the user of the image
id: the ID of the image 
tag: the emotion of the image (amusement, awe, contentment, excitement, anger, disgust, fear and sadness)
url: the download URL of the image
